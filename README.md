## Project Overview

This project analyzes customer demographics and purchase behavior to understand product preference across three Aerofit treadmill models.

## Dataset

The dataset includes:

- Age
- Gender
- Income
- Fitness Level
- Usage Frequency
- Product Purchased

## Tools Used

- Python
- Pandas
- NumPy
- Statistics
- Exploratory Data Analysis

## Key Insights

• Customers aged 21–30 showed the highest purchase probability for entry-level models.  
• Higher income groups showed preference for premium treadmills.  
• Fitness level strongly influenced product selection.

## Business Recommendations

• Target entry-level models toward younger customers.  
• Market premium treadmills to higher income segments.  
• Use fitness level as a key segmentation variable in marketing campaigns.
